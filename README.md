# コーディング課題（Figmaデザイン再現）

## 概要

提供されたFigmaのデザインデータを元に、
コーディング課題としてWebページを制作しました。
デザインの意図を汲み取り、見た目の再現を重視して実装しています。

## 制作目的

- デザインデータを読み取り、HTML / CSS に正確に落とし込む力の確認
- レイアウト構造や余白、文字組などの再現性を意識したコーディング

## 制作期間

約1週間

## 使用技術

- HTML
- CSS
- JavaScript
- jQuery

## 実装のポイント

- セマンティックな構造を意識したHTMLマークアップ
- デザインカンプに基づいたレイアウト・配色・タイポグラフィの再現
- スクロール位置に応じたUI制御（トップへ戻るボタン）
- フォーム要素の実装およびJavaScriptによる挙動確認

## 備考

本作品は学習課題として制作し、講評を受けたうえで完成としています。
